# Agent Behavior Experiments

## Experiment 1: Planning Before Execution
Prompted the agent to produce a plan before writing code.
Result: Responses became more structured and aligned with intent.

## Experiment 2: Verification Enforcement
Asked the agent to include verification steps.
Result: Reduced assumptions and clearer completion criteria.

## Experiment 3: Rule Refinement
Adjusted rules after observing overly rigid behavior.
Result: Improved balance between control and flexibility.
